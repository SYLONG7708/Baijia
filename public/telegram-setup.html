Telegram 結果群零基礎設定教學

返回首頁

把最高前 2 名預測提醒推送到 Telegram「結果群」

這份教學從零開始，帶你建立 Telegram Bot、建立群組、把 Bot 加進群組，最後把 Bot Token 填進 Baijia Pro 的 .env。完成後，首頁顯示的全桌平均分數最高前 2 名會自動傳到 Telegram。

Telegram Bot API 不能由程式直接幫你在手機建立私人群組。群組必須先由你在 Telegram 內建立， Baijia Pro 會在 Bot 加入群組後自動偵測並推送提醒。

你需要準備什麼

手機 Telegram

確認你可以正常登入 Telegram，並可以建立群組與搜尋 BotFather。

Baijia Pro 電腦

目前專案位置是 C:\Users\Administrator\Baijia，本機首頁是 http://localhost:4173/。

第一部分：用 BotFather 建立 Bot

打開 Telegram 並搜尋 BotFather

在 Telegram 搜尋列輸入 BotFather，選擇有 Telegram 官方驗證標記的 BotFather。

開始建立 Bot

進入 BotFather 對話後，輸入 /start，再輸入 /newbot。

輸入 Bot 顯示名稱

BotFather 會要求你輸入名稱。可以輸入 Baijia 結果提醒。

輸入 Bot 帳號名稱

帳號名稱必須以 bot 結尾，例如 baijia_result_7708_bot。如果名稱被使用，換一個即可。

複製 Token

成功後 BotFather 會給你一串 Token，格式通常像 123456789:AA...。這串就是 TELEGRAM_BOT_TOKEN。

Token 等同 Bot 密碼，不要貼到公開聊天、GitHub、截圖或直播畫面。

第二部分：建立 Telegram 群組「結果群」

在 Telegram 建立新群組

手機 Telegram 點選新增訊息，選擇「新增群組」。

群組名稱輸入結果群

群組名稱請輸入 結果群。如果你改成其他名稱，也要同步修改 TELEGRAM_GROUP_NAME。

把剛建立的 Bot 加入群組

進入群組資訊，選擇新增成員，搜尋你剛建立的 Bot 帳號，加入群組。

在群組傳一句話

傳送 測試 或任意文字。這一步很重要，Baijia Pro 才能用 Bot API 找到這個群組。

第三部分：把 Token 填進 .env

在 Windows 開啟檔案：

C:\Users\Administrator\Baijia\.env

在檔案底部加入或修改以下內容：

TELEGRAM_ENABLED=true
TELEGRAM_BOT_TOKEN=把BotFather給你的Token貼在這裡
TELEGRAM_GROUP_NAME=結果群
目前推送規則已取消百分比門檻，系統會取所有有效桌台的綜合平均分數最高前 2 名。

第四部分：重啟 Baijia Pro

修改 .env 後要重啟背景服務。打開 PowerShell，依序輸入：

cd C:\Users\Administrator\Baijia
if (Test-Path data\daemon.pid) { $pidNum = Get-Content data\daemon.pid; Stop-Process -Id $pidNum }
powershell -ExecutionPolicy Bypass -File scripts\start-24h.ps1

等待 10 到 20 秒後，打開首頁：

http://localhost:4173/

第五部分：確認是否成功

你看到的狀態 意思 處理方式
Telegram WATCHING 設定成功，正在等待資料更新或上榜提醒變化。 不用操作。
Telegram ALERT_READY 目前有符合條件的提醒，系統會推送到 Telegram。 檢查「結果群」是否收到訊息。
WAITING_BOT_TOKEN .env 沒有填 Bot Token。 回到第三部分，把 TELEGRAM_BOT_TOKEN 補上。
WAITING_GROUP Bot 尚未找到名稱為「結果群」的群組。 確認群組名稱、Bot 是否已加入群組，並在群組傳一句話。
Telegram ERROR Token 錯誤、網路問題、或 Bot 沒有權限。 檢查 Token 是否完整，並確認 Bot 在群組內。

如果自動找不到群組 ID

通常只要 Bot 加入「結果群」並傳一句話，系統會自動找到群組。如果找不到，可以手動取得 TELEGRAM_CHAT_ID。

用瀏覽器打開 getUpdates

把下面網址的 你的BOT_TOKEN 換成 BotFather 給你的 token：

https://api.telegram.org/bot你的BOT_TOKEN/getUpdates
找到 title 是結果群 的 chat id

頁面裡會看到類似內容：

"chat":{"id":-1001234567890,"title":"結果群"}
把 id 填回 .env 
TELEGRAM_CHAT_ID=-1001234567890

儲存後重啟 Baijia Pro。

推送規則

會推送

所有有效桌台中，綜合平均分數最高的前 2 名，且提醒內容有變化。

不會推送

資料校驗 ERROR、沒有有效資料、或提醒內容跟上一則完全相同。

Telegram 訊息會包含時間、桌號、預測莊/閒、平均分數與連續命中提醒。這是統計提醒，不保證下一局結果。

快速檢查網址

設定後可用這些網址確認狀態：

http://localhost:4173/
http://localhost:4173/api/alerts
http://localhost:4173/api/monitor

完成以上設定後，Baijia Pro 會 24 小時監控資料。只要首頁最高前 2 名提醒有變化， Telegram「結果群」就會收到同步通知。
