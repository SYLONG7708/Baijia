目標：電腦關機後仍 24 小時紀錄

從零開始註冊 Oracle Cloud，建立 Always Free Ubuntu 主機，部署 Baijia Pro

這份教學帶你完成 Oracle Cloud 註冊、建立免費 Ubuntu VM、開放 Baijia Pro 連接埠、 SSH 登入主機、執行一鍵安裝腳本。完成後，Baijia Pro 會用 systemd 常駐在雲端， 本機電腦與 Codex 關閉後仍可繼續記錄。

Oracle 註冊通常需要手機與信用卡/簽帳金融卡做身分驗證。請只建立標示 Always Free 或免費額度內的資源， 不要建立付費 Shape、付費磁碟、付費 Load Balancer，避免產生費用。

開始前先準備

需要的資料

  • 可收信的 Email。
  • 可收簡訊的手機號碼。
  • 可通過驗證的信用卡或簽帳金融卡。
  • 一台 Windows 電腦，用來 SSH 連線。

建議的 Oracle VM

  • 系統：Ubuntu 24.04 或 Ubuntu 22.04。
  • Shape：VM.Standard.A1.Flex。
  • 規格：先用 2 OCPU / 12 GB RAM。
  • Boot Volume：80 GB 以內。
Oracle 官方說明：Always Free Compute 必須在 Home Region 建立；Ampere A1 免費額度等同最多 4 OCPU 與 24 GB RAM，Block Volume Always Free 額度為 200 GB。

第一部分：註冊 Oracle Cloud Free Tier

打開 Oracle Free Tier 官方頁面

到 https://www.oracle.com/cloud/free/，點選 Start for free 或 Create a Free Account。

填寫國家、姓名、Email

Country 選你實際所在國家或地區，姓名與地址後面要一致。Email 請用常用信箱，Oracle 會寄驗證信。

完成真人驗證

點選 I am human，完成 CAPTCHA 驗證後按 Verify my email。

到信箱點驗證連結

打開 Oracle 寄來的信，點擊驗證連結。官方文件說驗證連結有時間限制，收到後請立刻完成。

設定 Cloud Account Name

這是你的 Oracle 租戶名稱，可用英文與數字，例如 baijia7708。之後登入 Oracle Console 會用到。

選 Home Region

Home Region 很重要。Always Free Compute 要在 Home Region 建立。建議優先選日本、韓國、新加坡等離你較近的區域；如果 A1 容量不足，只能在該區域等待或改 Availability Domain。

填寫地址與手機

地址、國家、電話請填真實資料，避免身分驗證失敗。手機驗證時輸入簡訊驗證碼。

填寫付款卡做驗證

Oracle 會用卡片做身分驗證。Free Tier 頁面說卡片資訊用於帳號設定與身分驗證；請不要升級或建立付費資源。

等待帳號建立完成

完成後會進入 Oracle Cloud Console，或收到登入網址。登入頁常見網址是 https://cloud.oracle.com/。

第二部分：建立 Always Free Ubuntu VM

進入 Compute Instances

Oracle Console 左上角選單，進入 Compute，再點 Instances。

點 Create instance

名稱輸入 baijia-pro-24h。Compartment 沿用預設即可。

Image 選 Ubuntu

點 Change image，選 Canonical Ubuntu，版本用 Ubuntu 24.04 或 22.04。

Shape 選 Ampere A1

點 Change shape，選 Virtual machine，Processor 選 Ampere，Shape 選 VM.Standard.A1.Flex。

先設定 2 OCPU 與 12 GB RAM。如果顯示容量不足，改成 1 OCPU / 6 GB RAM 或換 Availability Domain 再試。

Networking 保留 Public IPv4

可以讓 Oracle 自動建立 VCN。確認有勾選 Assign a public IPv4 address，否則你的瀏覽器無法直接連到 Baijia 頁面。

設定 SSH Key

如果你沒有 SSH Key，可以用 Oracle 的 Generate a key pair for me，下載 Private Key 與 Public Key。Private Key 要保存好，不能外流。

如果你想自己在 Windows 產生，打開 PowerShell：

ssh-keygen -t ed25519 -C "baijia-oracle"

一路按 Enter，然後把 C:\Users\Administrator\.ssh\id_ed25519.pub 的內容貼到 Oracle。

Boot volume 不要超額

Boot volume 建議 80 GB，不要超過 Always Free Block Volume 總額。建立前確認畫面上沒有付費警告。

建立 Instance

按 Create。建立完成後，複製 Public IP，例如 129.xxx.xxx.xxx。

第三部分：開放 Baijia Pro 連接埠 4173

Oracle 預設只開 SSH 22。Baijia Pro 網頁需要 TCP 4173。

進入 VCN Security List

在 Instance 詳細頁找到 VCN，點進去後找 Security Lists 或 Network Security Groups。

新增 Ingress Rule

新增一條規則：

Source CIDR建議填你的固定 IP。沒有固定 IP 才填 0.0.0.0/0
IP ProtocolTCP
Destination Port Range4173
DescriptionBaijia Pro dashboard
如果你把 Source CIDR 設成 0.0.0.0/0，全世界都能看到你的 Baijia 首頁。更安全的做法是只允許自己的 IP，或之後加 Nginx 密碼保護。

第四部分：從 Windows 登入 Oracle VM

打開 Windows PowerShell，使用 Oracle Instance 的 Public IP 登入。Ubuntu 預設使用者通常是 ubuntu。

ssh ubuntu@你的Oracle公網IP

如果你使用 Oracle 下載的 Private Key：

ssh -i C:\Users\Administrator\Downloads\ssh-key-xxxx.key ubuntu@你的Oracle公網IP

第一次連線會問是否信任主機，輸入 yes。

第五部分：一鍵安裝 Baijia Pro

SSH 登入後，直接貼上這條命令：

curl -fsSL https://raw.githubusercontent.com/SYLONG7708/Baijia/main/scripts/install-oracle-ubuntu.sh | bash

安裝腳本會自動完成：

  • 安裝 Node.js 24。
  • 下載 GitHub 上的 Baijia Pro。
  • 安裝 npm 套件與 Playwright Chromium。
  • 建立 /etc/baijia/baijia.env。
  • 建立 systemd 服務 baijia-pro。
  • 設定開機自動啟動。

第六部分：填入 Allbet 與 Telegram 設定

安裝完成後，編輯雲端主機上的環境檔：

sudo nano /etc/baijia/baijia.env

至少填入：

ALLBET_URL=https://www.abgame88.net/?hallView=all&loginType=2&language=zh_TW&sessionId=你的sessionId
TELEGRAM_ENABLED=true
TELEGRAM_BOT_TOKEN=你的TelegramBotToken
TELEGRAM_CHAT_ID=你的群組ID
TELEGRAM_GROUP_NAME=結果群
PUBLIC_API_BASE=http://你的Oracle公網IP:4173

存檔方式：按 Ctrl + O，Enter，然後 Ctrl + X 離開。

重啟服務：

sudo systemctl restart baijia-pro

第七部分：確認 24H 服務正常

檢查項目 命令或網址 正常結果
systemd 狀態 sudo systemctl status baijia-pro --no-pager 看到 active (running)
本機 API curl http://127.0.0.1:4173/api/monitor 看到 JSON，且 monitor / scraper 有狀態
瀏覽器頁面 http://你的Oracle公網IP:4173/ 能打開 Baijia Pro 首頁
即時日誌 sudo journalctl -u baijia-pro -f 沒有連續錯誤，資料持續更新

常見問題排除

問題 原因 處理方式
註冊卡在付款驗證 卡片、地址、電話或地區資料不一致 改用真實資料與可線上刷卡的卡片；不要用 VPN 或不一致地區資料
A1 建立失敗，Out of host capacity 該區域暫時沒有 Always Free A1 容量 換 Availability Domain、降低到 1 OCPU / 6 GB，或晚點再試
網頁打不開 沒有開 4173，或 Public IP 錯誤 檢查 Oracle Ingress Rule：TCP 4173；確認網址是 http://IP:4173
服務沒有資料 ALLBET_URL 沒填，或 session 失效 重新取得 Allbet 遊戲網址，更新 /etc/baijia/baijia.env 後重啟
Telegram 沒收到 Token 或 Chat ID 錯誤 確認 bot 已加入「結果群」，群內有傳過訊息，並填入正確 TELEGRAM_CHAT_ID
